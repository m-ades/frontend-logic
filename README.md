# PHILO 275 Classroom

LMS for PHILO 275 at Hunter College.

## Tech Stack

- **React**
- **Material UI**
- **Vite**

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create an `.env` file with your x-api-key.

3. Start the development server:
```bash
npm run dev
```
